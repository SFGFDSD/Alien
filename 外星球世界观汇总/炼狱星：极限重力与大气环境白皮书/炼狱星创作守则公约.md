# 极限星球生物设定基准（修订版）

## 核心起源设定

\> **本星球所有动物均起源于同一祖先：内外双骨骼生物**

\>

\> 同时具备外骨骼（承压抗冲击）+ 内骨骼（支撑内脏、肌肉附着点）

---

## 第一部分：星球物理环境

| 参数 | 数值 | 对比地球 |

|------|------|----------|

| 表面重力 | 167 m/s² | 17g |

| 大气密度 | 120 kg/m³ | 100倍 |

| 大气压强 | 150 atm | 150倍 |

| 等效生物承受重力 | ~15g | 浮力补偿后 |

| 氧气分压 | 0.22 atm | 氧含量0.15% |

| 地表光照 | 35% | 昏暗 |

## 第二部分：双骨骼系统详解

### 2.1 结构原理

```

横截面示意图：

┌──────────外骨骼层──────────┐

│ ╔══════════════════════╗ │

│ ║ 肌肉层 ║ │

│ ║ ┌─────内骨骼─────┐ ║ │

│ ║ │ 内脏腔 │ ║ │

│ ║ │ ●肝 ●肠 ●腺 │ ║ │

│ ║ └─────────────────┘ ║ │

│ ╚══════════════════════╝ │

└────────────────────────────┘

```

### 2.2 双骨骼功能分工

| 结构 | 材质 | 功能 |

|------|------|------|

| **外骨骼** | 几丁质+ ite复合 | 承受外压、抗冲击、防护 |

| **内骨骼** | 钙化软骨 | 支撑内脏、肌肉附着、维持形态 |

| **连接结构** | 纤维韧带 | 双骨骼间力传导 |

### 2.3 演化优势

| 优势 | 说明 |

|------|------|

| 压力分散 | 外骨骼承接17g重力，内骨骼保护器官 |

| 冗余保护 | 外壳破裂时内骨骼仍可短暂维持生命 |

| 力量传导 | 肌肉夹于双层间，效率更高 |

| 器官悬挂 | 内脏挂于内骨骼，减少晃动损伤 |

## 第三部分：生物体型绝对上限

### 3.1 各生态位体型极限

| 生态位 | 最大长 | 最大宽 | 最大高 | 体重上限 | 形态约束 |

|--------|--------|--------|--------|----------|----------|

| **陆生顶级动物** | 8 cm | 4 cm | 1.5 cm | 35 g | 扁平+多足 |

| **陆生掠食者** | 5 cm | 2.5 cm | 1 cm | 10 g | 快速扁平型 |

| **陆生常见动物** | 2-3 cm | 1-1.5 cm | 0.5 cm | 2 g | |

| **微型陆生** | 2-5 mm | 1-2 mm | 0.5 mm | <10 mg | 双骨骼退化 |

| **水生最大** | 50 cm | 15 cm | 10 cm | 5 kg | 浮力补偿 |

| **水生常见** | 10-20 cm | 5 cm | 3 cm | 100 g | |

| **气浮型** | 20 cm | 20 cm | 5 cm | 50 g | 外骨骼轻量化 |

| **附着型** | 50 cm(铺展) | 50 cm | 0.3 cm | 不限 | 无结构应力 |

| **穴居型** | 15 cm | 1 cm | 1 cm | 8 g | 外骨骼厚化 |

### 3.2 禁止存在的生物形态

| 形态 | 原因 |

|------|------|

| ✗ 体高>3cm的陆生生物 | 双骨骼也无法对抗血液压差 |

| ✗ 细长腿部 | 即使双层骨骼也会断裂 |

| ✗ 单骨骼大型生物 | 无法承受15g等效重力 |

| ✗ 跳跃型生物 | 落地冲击会压碎双层结构 |

## 第四部分：强制性生理特征

### 4.1 双骨骼结构要求

| 部位 | 外骨骼特征 | 内骨骼特征 |

|------|------------|------------|

| **背甲** | 厚实拱形 | 横向肋条 |

| **腹部** | 略薄、分节 | 纵向脊柱 |

| **足部** | 管状硬壳 | 中心骨棒 |

| **头部** | 盔状 | 脑腔支架 |

### 4.2 循环系统

| 特征 | 说明 |

|------|------|

| 半开放式血腔 | 内骨骼间形成血液通道 |

| 多心室分布 | 3-6个小心室水平排列 |

| 高粘度血液 | 防止15g下快速下沉 |

| 最大垂直距离 | <1.5cm（内骨骼维持腔室高度） |

### 4.3 呼吸系统

| 特征 | 说明 |

|------|------|

| 外骨骼气孔 | 分布于侧面，可开闭 |

| 书肺 | 内骨骼支撑的薄片结构 |

| 高亲和氧载体 | 适应0.15%低氧环境 |

### 4.4 神经系统

| 特征 | 说明 |

|------|------|

| 分布式神经节 | 沿内骨骼分布 |

| 主神经索 | 沿内骨骼脊柱走行 |

| 无垂直脑 | 神经节水平排列 |

## 第五部分：环境行为约束

### 5.1 运动方式

| 允许 | 禁止 |

|------|------|

| ✓ 贴地爬行 | ✗ 跳跃 |

| ✓ 滑行 | ✗ 奔跑 |

| ✓ 稠密大气中"游泳" | ✗ 扑翼飞行 |

| ✓ 蠕动 | ✗ 直立行走 |

| ✓ 缓慢攀爬 | ✗ 快速攀爬 |

### 5.2 速度限制

| 体型 | 最大持续速度 |

|------|--------------|

| 8cm陆生 | ~0.5 m/s |

| 3cm陆生 | ~0.3 m/s |

| 水生 | ~2 m/s |

### 5.3 坠落规则

| 高度 | 结果 |

|------|------|

| <5 cm | 安全（双骨骼+大气缓冲） |

| 5-20 cm | 外骨骼裂纹 |

| 20-50 cm | 外骨骼破裂、内骨骼骨折 |

| >50 cm | 双层结构粉碎、致命 |

## 第六部分：双骨骼演化分支

```

┌─→ 外骨骼强化型（厚甲慢行）

│

原始双骨骼祖先 ──┼─→ 均衡型（主流形态）

│

├─→ 内骨骼强化型（水生方向）

│

└─→ 双层轻量化（气浮型）

```

| 分支 | 特征 | 典型生态位 |

|------|------|------------|

| **外骨骼强化型** | 厚壳、内骨骼退化 | 穴居、防御型 |

| **均衡型** | 双层均衡 | 陆生主流 |

| **内骨骼强化型** | 外壳薄化、内骨骼发达 | 水生大型 |

| **双层轻量化** | 两层均薄、多气囊 | 气浮漂浮型 |

## 第七部分：创作合规检查表

设计新生物时逐项确认：

| 检查项 | 要求 |

|--------|------|

| □ 双骨骼 | 必须具备（微型可退化） |

| □ 体高 | ≤1.5cm（陆生）/ ≤10cm（水生） |

| □ 体长 | ≤8cm（陆生）/ ≤50cm（水生） |

| □ 外形 | 扁平度 ≥3:1 |

| □ 足数 | ≥12（若有足） |

| □ 循环 | 半开放+多心室 |

| □ 神经 | 沿内骨骼分布 |

| □ 运动 | 无跳跃/无直立 |

| □ 速度 | ≤0.5m/s（陆生） |

---

**所有生物创作必须遵守双骨骼起源设定与物理定律约束。**
